## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

- [Simple, fast routing engine](https://laravel.com/docs/routing).
- [Powerful dependency injection container](https://laravel.com/docs/container).
- Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
- Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
- Database agnostic [schema migrations](https://laravel.com/docs/migrations).
- [Robust background job processing](https://laravel.com/docs/queues).
- [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

Loyiha haqida

81-Maktab — o'quvchilar, o'qituvchilar, ota-onalar va administratorlar uchun mo'ljallangan veb-asosli ta'lim platformasi. Loyiha maktab yangiliklarini e'lon qilish, o'qituvchi profillarini yuritish, kurslar ochish, onlayn imtihon o'tkazish, ichki muloqot (chat) va sun'iy intellekt yordamida savol-javob berish kabi imkoniyatlarni bitta tizimga jamlaydi.

O'rnatish

Talablar: PHP 8.2+, Composer, Node.js 18+, MySQL, (ixtiyoriy: Redis)

bash# 1) Repozitoriyani klonlash
git clone https://github.com/xabibullayevshamsiddin/81-maktab.git
cd 81-maktab

# 2) PHP va JS bog'liqliklarini o'rnatish
composer install
npm install

# 3) .env faylini yaratish va ilova kalitini generatsiya qilish
cp .env.example .env
php artisan key:generate

# 4) .env faylida ma'lumotlar bazasi sozlamalarini kiriting
#    DB_DATABASE, DB_USERNAME, DB_PASSWORD

# 5) Migratsiya va boshlang'ich ma'lumotlar (rollar + AI bilim bazasi)
php artisan migrate --seed

# 6) Fayllar (rasmlar) uchun storage havolasini yaratish
php artisan storage:link

# 7) Frontend qurish
npm run dev      # development rejimida
# yoki
npm run build     # production uchun

# 8) Serverni ishga tushirish
php artisan serve

Ilova endi http://localhost:8000 manzilida ishlaydi.

Test hisoblar (ixtiyoriy): lokal muhitda tezroq sinab ko'rish uchun tayyor admin/editor/moderator hisoblarini yaratish mumkin:

bashphp artisan db:seed --class=AdminUserSeeder


⚠️ Bu buyruq oddiy, taxmin qilish oson parollar bilan hisoblar yaratadi (masalan admin123). Faqat lokal muhitda ishlating, productionda hech qachon ishga tushirmang va agar ishlatsangiz, parollarni darhol almashtiring.

Joylashtirish (Deploy)

Loyihada bir nechta joylashtirish varianti tayyorlangan:

Docker + Render.com — Dockerfile va render.yaml orqali. Deploy vaqtida avtomatik migrate, route:cache, view:cache bajariladi. Eng izchil va to'liq sozlangan variant.

bashdocker build -t 81-maktab .
docker run -p 8080:8080 --env-file .env 81-maktab

Nixpacks (masalan Railway) — nixpacks.toml orqali PHP muhiti avtomatik aniqlanadi.

Vercel — vercel.json fayli mavjud, biroq hozirgi holatda to'liq sozlanmagan (funksiya kirish nuqtasi qismida nomuvofiqlik bor) — ishlatishdan oldin sinab ko'ring.

Productionga chiqarishdan oldin: APP_ENV=production, APP_DEBUG=false va barcha kerakli API kalitlarini (GEMINI_API_KEY, to'lov va email kalitlari) muhit o'zgaruvchilariga kiriting.
